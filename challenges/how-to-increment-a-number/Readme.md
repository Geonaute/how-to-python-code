## Challenge Description
Take a number as user input and perform the following operations once.

- If the number is odd add 1
- If the number is even add 3
- If the number is divisible by 5 add 5

## Example
```
input: 2
output: 5

input: 3
output: 4

input: 5
output: 11
```
